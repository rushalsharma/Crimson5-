# Crimson 5 Energy Cloud Management Simulator

Crimson 5 Energy Cloud Management Simulator is a Streamlit-based capstone dashboard that brings financial planning, governance, delivery readiness, resilience design, AI value assessment, adoption dynamics, and market diffusion into one integrated executive view. The app is designed for presentation and scenario exploration, allowing users to adjust assumptions and interpret strategy-level signals across cloud transformation dimensions.

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dashboard Sections

- Panels 1–9
- Cloud Cost Explorer
- FinOps Insights
- AI Investment Demo

## Demo Note

AWS cost data runs in demo mode unless credentials are entered.
